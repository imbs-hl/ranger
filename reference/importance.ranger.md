# ranger variable importance

Extract variable importance of ranger object.

## Usage

``` r
# S3 method for class 'ranger'
importance(x, ...)
```

## Arguments

- x:

  ranger object.

- ...:

  Further arguments passed to or from other methods.

Variable importance measures.
