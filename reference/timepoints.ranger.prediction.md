# Ranger timepoints

Extract unique death times of Ranger Survival prediction object.

## Usage

``` r
# S3 method for class 'ranger.prediction'
timepoints(x, ...)
```

## Arguments

- x:

  Ranger Survival prediction object.

- ...:

  Further arguments passed to or from other methods.

## Value

Unique death times
